<h1> Finance Tracker </h1>
<p> A full-stack data science project that tracks personal finances by ingesting bank account transaction data, categorizes spending, stores it in a SQL database, and visualizes income vs. expenses through an interactive Tableau dashboard.</p>

<li>🏗️ Currently in the process of creating </li>
